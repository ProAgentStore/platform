About ProAgentStore

ProAgentStore is a platform for server-powered AI agents that need more than the browser can give. Workers AI for 70B+ parameter models, D1 databases for persistent state, R2 for file storage, cron for scheduling, and an API gateway to expose your agent as a service. $9/mo for everything.

Why server-side?

FreeAgentStore runs AI in the browser -- WebGPU, WASM, heuristics. That's great for privacy and cost. But some agents need more:

Bigger models. Browser caps out around 2-3GB. Server-side Workers AI runs Llama 3, Mistral, Whisper Large, SDXL -- any model, any size.

Persistent state. Browser storage is ephemeral. A D1 database survives across sessions, devices, and users. Your agent remembers everything.

Background work. Cron scheduling lets agents run on a schedule -- daily digests, monitoring, batch processing -- without anyone visiting the page.

API access. Expose your agent as a REST endpoint. Other apps, other agents, webhooks -- anything can call your agent programmatically.

What each agent gets

Workers AI

Server-side inference. Any model from Cloudflare's catalog -- LLMs, image generation, speech, embeddings.

D1 Database

Per-agent SQLite. Conversations, knowledge bases, user prefs, analytics. Survives forever.

R2 Storage

Per-agent file storage. Generated audio, images, documents, exports. Served via CDN.

Cron

Scheduled execution. Daily reports, monitoring checks, data pipelines, notification digests.

API Gateway

REST endpoint with auth + rate limits. Other apps call your agent as a service.

Stripe Payouts

Earn from the $9/mo pool. Usage-weighted splits via Stripe Connect. Spotify model.

Pricing

One subscription: $9/mo. That unlocks every Pro agent on the platform. No per-agent pricing, no usage billing, no surprises.

Creators earn from the subscription pool in proportion to their agent's share of usage each month, minus a 10% platform fee. Same model as Spotify -- the more people use your agent, the more you earn.

Free vs Pro

They're different products, not tiers. FreeAgentStore runs AI on your device -- private, offline-capable, zero cost. ProAgentStore runs AI on Cloudflare's edge -- powerful, persistent, connected. Start free, graduate to Pro when you need server-side compute.
