Get Started

Three ways to use ProAgentStore: try an agent, subscribe to one, or build your own.

Try an agent

1

Browse the store

Visit proagentstore.online and pick an agent. Each one has a live trial — no sign-up needed.

2

Chat in the detail page

Every agent detail page has a "Try it now" chat box. 20 free messages per session.

Subscribe to an agent

1

Sign in with GitHub

Go to the Console and sign in. One click — no separate account needed.

2

Get the agent

Click "Get Agent" on any store listing. You get your own isolated instance of the agent.

3

Add your documents

Upload your files, paste text, or import URLs. The agent answers based on YOUR data. Private to you — the creator never sees it.

4

Chat or call the API

Use the console chat, or integrate via the REST API and SDK.

import { initPro } from '@proagentstore/sdk'

const agent = initPro({
  agentId: 'your-instance-id',
  token: 'your-token',
})

const { response } = await agent.chat('Summarize my Q3 report')
console.log(response)

Build your own agent

1

Install the CLI

Scaffold a complete agent project with one command.

npx @proagentstore/cli init my-agent --template worker

Three templates:

  • worker — Durable Object with conversation, memory, knowledge base, 10 tools
  • cron — Scheduled worker (daily digests, monitoring, batch processing)
  • api — Stateless endpoint (transform, generate, analyze)
2

Customize

Edit src/index.ts — set the system prompt, add knowledge docs, wire up tools. Or use the Console UI to configure without code.

3

Publish

Run compliance checks and publish to the store in one command.

npx @proagentstore/cli check    # 11-point compliance check
npx @proagentstore/cli publish  # creates repo, deploys, registers

Your agent appears at proagentstore.online/agents/my-agent/ with a live trial, subscribe button, and embed code.

Embed on your site

Add an agent as a chat widget on any website:

<script src="https://proagentstore.online/widget.js"
  data-agent="your-agent-slug"
  data-theme="dark"></script>

Floating chat button, expandable panel, dark/light theme. Works on any origin.

Connect via webhook

Send documents from Zapier, Make, n8n, or any service:

curl -X POST https://api.proagentstore.online/v1/public/webhook/INSTANCE_ID/ingest \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"title":"Meeting Notes","content":"...","source":"zapier"}'

MCP integration

Manage agents from Claude Code, Cursor, or VS Code:

npx mcp-remote https://mcp.proagentstore.online/mcp

Use chat_with_agent only for public trial chats. For the real user runtime, call subscribe_agent, then my_instances, then chat_with_instance. Instance tools keep user knowledge and chat state separate from the creator's template agent.

Store your API keys

Use your own OpenAI, Anthropic, or Google AI keys for higher-quality responses. Keys are encrypted with AES-256-GCM and injected server-side via proxy — never exposed to agents.

Go to Console → Profile → API Keys.

Supported: OpenAI, Anthropic, Google AI (Gemini), OpenRouter, Groq, Together AI.

Browser-side agents?

If your agent runs in the browser (no server needed), use FreeAgentStore instead — free forever, fully private, offline-capable. ProAgentStore is for agents that need server power: cron, databases, large models, webhooks, multi-user state.
