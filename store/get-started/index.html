Get Started

Build a server-powered AI agent and deploy it to Cloudflare. Takes 10 minutes.

Use an existing Pro agent

1

Subscribe

$9/mo unlocks every Pro agent. One subscription, all agents. Browse the catalog to see what's available.

2

Call the agent's API

Every Pro agent has a REST endpoint. Authenticate with your subscription token and call it from anywhere.

curl https://api.proagentstore.online/agents/summarizer/run \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"text": "Long article to summarize..."}'

Build a new Pro agent

1

Install the CLI

The CLI scaffolds your agent, runs it locally, and deploys to Cloudflare.

pnpm add -g @proagentstore/cli
2

Create the agent

Pick a template. Each template includes Workers AI, D1, and R2 pre-wired.

pags init my-agent --template ai-chat
cd my-agent
3

Write the logic

Your agent is a Cloudflare Worker. Use the SDK to access AI, database, storage, and more.

import { initPro } from '@proagentstore/sdk'

const agent = initPro({ agentId: 'my-agent' })

export default {
  async fetch(request, env) {
    const { prompt } = await request.json()

    // Run a 70B model server-side
    const response = await agent.ai.run('@cf/meta/llama-3.3-70b-instruct-fp8-fast', {
      messages: [{ role: 'user', content: prompt }]
    })

    // Save to persistent database
    await agent.db.exec(
      'INSERT INTO conversations (prompt, response) VALUES (?, ?)',
      prompt, response.text
    )

    return Response.json({ text: response.text })
  },

  // Cron: runs daily at 8am UTC
  async scheduled(event, env) {
    const summary = await agent.ai.run('@cf/meta/llama-3.2-3b-instruct', {
      messages: [{ role: 'user', content: 'Summarize yesterday\'s conversations' }]
    })
    await agent.storage.put('daily-summary.txt', summary.text)
  }
}
4

Test locally

The CLI runs your agent with a local D1 database and mock AI responses.

pags dev
5

Publish

Push to the ProAgentStore org. CI deploys the Worker, provisions D1 + R2, sets up the subdomain and API endpoint.

pags publish

Your agent is live at my-agent.proagentstore.online with a REST API at api.proagentstore.online/agents/my-agent/.

Upgrading from Free

Already have a browser agent on FreeAgentStore? Pro agents can import free agents as dependencies -- keep the browser-side logic, add server-side compute on top.

import { parseResume } from '@freeagentstore/resume-parser'
import { initPro } from '@proagentstore/sdk'

const agent = initPro({ agentId: 'smart-recruiter' })

// Use the free heuristic parser + Pro AI for analysis
const parsed = parseResume(resumeText)
const analysis = await agent.ai.run('@cf/meta/llama-3.3-70b-instruct-fp8-fast', {
  messages: [{ role: 'user', content: `Evaluate this candidate: ${JSON.stringify(parsed)}` }]
})

SDK reference

const agent = initPro({ agentId: 'my-agent' })

agent.ai          // Workers AI -- run any model server-side
agent.db          // D1 database -- per-agent SQLite
agent.storage     // R2 storage -- per-agent file bucket
agent.subscription // Stripe -- checkout, portal, status
agent.usage       // Usage tracking -- for creator payouts
