Launch + 11 Agents

Launch

ProAgentStore goes live with a full marketplace platform and 11 server-powered AI agents.

  • Platform: Store with search + sort, Console (creator + client), Admin dashboard, MCP server (10 tools), CLI (init/check/publish)
  • 11 Agents: chatbot, site-monitor, lead-qualifier, content-pipeline, competitor-intel, support-escalator, data-analyst, meeting-notes, seo-auditor, invoice-parser, email-drafter
  • Marketplace: Subscribe to agents, get isolated instances with your own data
  • Knowledge Base: Paste text, import URLs, upload files, Google Docs integration
  • Guardrails: Topic restrictions, blocked terms, response style, max length, citations
  • Key Vault: Store API keys for OpenAI, Anthropic, Google AI, OpenRouter, Groq, Together AI
  • Embeddable Widget: Add any agent as a chat widget on your site
  • Developer Profiles: Public pages with all published agents, cross-store links
