Support

Help for the native ProAgentStore app.

Getting Started

  • Install the iOS app from TestFlight or the App Store.
  • Tap Continue with ProAgentStore and sign in.
  • Open an agent instance to chat.
  • Use Board to review waiting tasks, approvals, cancellations, and completed work.
  • Use Coder to manage coding repositories and cloud coding sessions.
  • Use Settings to edit special instructions and profile context.

Common Issues

Sign in does not return to the app

Make sure the installed app is the current build and retry sign-in. The native app expects the pags://auth callback.

No instances appear

Confirm your ProAgentStore account has a private instance. You can create or subscribe to agents from the ProAgentStore console.

Tasks are missing from the board

The board only shows runtime tasks available to your signed-in account and selected instance. Pull to refresh or reopen the instance.

Coding sessions do not start

Confirm the repository is configured for the signed-in account and that the ProAgentStore coding runtime is available.
